- 20 Bytes long
- U: urgent
- P: push --push 给 application
- A: ack 如果 acknowledge number 是 valid 则为 1，发起连接前是 0
- R: reset
- S: syn 起始 sequence number，请同步到这个号码
- F: fin
- offset: 偏移，这是因为TCP 可能 options
